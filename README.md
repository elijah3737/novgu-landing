# От консультации к поступлению — НовГУ

Интерактивный обучающий лендинг для сотрудников приёмной комиссии Новгородского государственного университета.

## Запуск

Открыть `index.html` в браузере. Никаких зависимостей не нужно.

Или через локальный сервер:
```bash
python3 -m http.server 8771
# http://localhost:8771
```

## Структура

- `index.html` — лендинг (HTML + CSS + JS в одном файле)
- `TZ_v2.md` — техническое задание v2 с KPI, структурой, тренажёром, аттестацией

## Что внутри

- Выбор роли (горячая линия / очная ПК / институт / соцсети)
- Интерактивная воронка из 8 этапов
- 7 шагов правильного разговора
- Банк вопросов и возражений (5 вкладок)
- Тренажёр диалогов: 8 ситуаций с разбором «плохо / средне / хорошо»
- Этическая рамка: сопровождение vs давление
- Чек-лист с сохранением прогресса
- Печатная карточка на стол (PDF через `window.print`)
- Шаблон CRM-карты (CSV)
- Аттестация: 10 вопросов + именной сертификат
- Адаптив под десктоп / планшет / смартфон

## Технологии

Чистый HTML/CSS/JS без фреймворков. Прогресс сохраняется в localStorage. Контент тренажёра и аттестации вынесен в JS-объекты — легко обновлять.
